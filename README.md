# GoldPricePrediction
Gold Price Prediction Using Fuzzy Logic

**Tech Stack:** Python, TensorFlow, LSTM, Bi-LSTM, FastAPI, yFinance API, Skfuzzy, Pandas, NumPy, Matplotlib, HTML/CSS, Jinja2

**Project Description**

Developed an intelligent financial forecasting system for gold price prediction by combining fuzzy logic with deep learning techniques. The project utilized real-time historical market data collected from the Yahoo Finance (yFinance) API to build a robust time-series prediction pipeline. Multiple fuzzy membership functions including Gaussian, Trapezoidal, Triangular, and Sigmoid were integrated for handling uncertainty and nonlinear behavior in financial data. Deep learning architectures such as LSTM and Bi-LSTM were used to improve sequential learning and forecasting accuracy. The system also included data preprocessing, feature engineering, model evaluation, and visualization components for comprehensive prediction analysis. A FastAPI backend and interactive frontend dashboard were developed to support real-time forecasting and experimentation workflows.
